# Hand Gesture Mouse 🖐️🖱️

Control your computer using hand gestures with the help of Python, OpenCV, and MediaPipe.

---

## 🔥 Features
- 🖱️ Cursor movement using index finger
- 🤏 Left click using thumb + index finger
- ✌️ Right click using thumb + middle finger
- 📜 Scroll using finger movement
- 🎯 Smooth cursor control

---

## 🧠 How it works
This project uses computer vision to detect hand landmarks using MediaPipe.  
Then it maps finger movements to mouse actions using PyAutoGUI.

---

## ⚙️ Requirements
Install the following libraries:
pip install opencv-python mediapipe pyautogui

---

## 🎮 Controls

| Gesture | Action |
|--------|--------|
| Index finger move | Cursor move |
| Thumb + Index close | Left click / Drag |
| Thumb + Middle close | Right click |
| Index up/down | Scroll |
